# SGIME - Plugin de Tarefas Recorrentes

## Versão Customizada para SGIME v1.6

Este plugin foi adaptado especificamente para o Sistema de Gestão Integrada de Engenharia (SGIME) do Colégio Pedro II, com foco em tarefas de manutenção predial conforme ABNT NBR-5674.

### Características da Versão SGIME:

✅ **Compatível com Rails 7.2** - Removido `unloadable` deprecado  
✅ **Compatível com Redmine 6.0** - Migrações atualizadas  
✅ **Personalização SGIME** - Interface e terminologia adaptadas  
✅ **Foco em Manutenção Predial** - Configurações específicas para ABNT NBR-5674  

### Funcionalidades:

- **Tarefas Recorrentes**: Criação automática de tarefas de manutenção
- **Cronograma Flexível**: Suporte a intervalos diários, semanais, mensais e anuais  
- **Integração com Projetos**: Associação com projetos específicos de manutenção
- **Relatórios**: Visualização de tarefas programadas e histórico

### Instalação:

O plugin já está integrado ao docker-compose.yml do SGIME e será instalado automaticamente.

### Configuração:

1. Acesse **Administração** > **Plugins** > **SGIME - Tarefas Recorrentes**
2. Configure as opções específicas do sistema
3. Defina permissões de usuários para tarefas recorrentes

### Uso:

1. Em qualquer projeto, acesse **Tarefas Recorrentes**
2. Clique em **Nova Tarefa Recorrente**
3. Configure o cronograma de recorrência
4. Associe com uma tarefa existente ou crie uma nova
